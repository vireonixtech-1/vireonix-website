## Features

### SEO Optimized
- Schema.org structured data (Organization, LocalBusiness, FAQ, HowTo, BreadcrumbList)
- Open Graph & Twitter Card meta tags
- Automatic sitemap generation
- Optimized robots.txt

### AEO (Answer Engine Optimization)
- FAQ Schema for featured snippets
- Speakable content markup
- HowTo Schema for process visibility
- Structured Q&A content

### GEO (Generative Engine Optimization)
- `llms.txt` for AI crawler guidance
- `.well-known/ai-plugin.json` manifest
- AI-friendly content structure
- All major AI crawlers allowed (GPTBot, ClaudeBot, PerplexityBot, etc.)

### Performance
- Static site generation with Astro
- Optimized images
- Minimal JavaScript
- Mobile-first responsive design

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/vireonixtech-1/vireonix-website.git

# Navigate to project
cd vireonix-website

# Install dependencies
npm install

# Start development server
npm run dev
```

### Commands

| Command | Action |
|---------|--------|
| `npm run dev` | Start dev server at `localhost:4321` |
| `npm run build` | Build production site to `./dist/` |
| `npm run preview` | Preview build locally |
